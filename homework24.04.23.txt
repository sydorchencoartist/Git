1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
Чтобы скопировать репозиторий с GitHub на свой компьютер, нужно сначала скопировать ссылку на репозиторий. Для этого на странице репозитория на GitHub нужно нажать на кнопку "Clone or download" и скопировать ссылку. Затем на своем компьютере нужно открыть терминал, перейти в нужную директорию и выполнить команду "git clone" с указанием скопированной ссылки.

2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
Чтобы создать git репозиторий из локального каталога на своем компьютере, нужно в терминале перейти в нужную директорию и выполнить команду "git init".
Затем нужно добавить файлы в репозиторий, используя команду "git add" и создать коммит с описанием изменений, используя команду "git commit".
Чтобы отправить репозиторий на GitHub, нужно создать новый репозиторий на сайте GitHub, затем выполнить команду "git remote add" с указанием ссылки на репозиторий на GitHub и выполнить команду "git push".

3. Как получить на локальный компьютер изменения с github.com
Чтобы получить на локальный компьютер изменения с GitHub, нужно выполнить команду "git pull" в директории репозитория на локальном компьютере.

4. Какую информацию нам дают команда :
git status
git log
git remote -v
Команда "git status" показывает текущее состояние репозитория, включая измененные, добавленные или удаленные файлы и неотслеживаемые файлы.
Команда "git log" показывает историю коммитов в репозитории, включая их идентификаторы, даты, авторов и сообщения коммитов.
Команда "git remote -v" показывает список удаленных репозиториев, связанных с локальным репозиторием, и их адреса.
